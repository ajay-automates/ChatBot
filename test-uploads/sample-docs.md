# Product Documentation

## Getting Started

To get started with our platform, first create an account by visiting our signup page. Once your email is verified, you'll have access to the dashboard where you can create your first project.

## Creating a Project

Navigate to the Projects tab and click "New Project". Give your project a name and select your preferred settings. Projects can be customized with different themes and configurations.

## Inviting Team Members

From your project dashboard, click on the "Team" tab. Enter the email addresses of team members you want to invite. They'll receive an invitation email with instructions to join.

## API Integration

Our platform provides a RESTful API for integration. You can find your API key in the Settings > API section. Use this key in the Authorization header of your requests.

## Billing and Subscriptions

Manage your subscription from the Billing section. You can upgrade, downgrade, or cancel at any time. All changes take effect immediately, and we prorate charges accordingly.
